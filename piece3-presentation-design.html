Slide grid — 12 column
Scalable 12-column grid adapted for both 16:9 widescreen and letter-format print exports.
Brief
Design a reusable executive presentation system for a mid-size consulting firm — scalable across quarterly reviews, client pitches, and internal strategy updates. Must work for both screen and print.
Approach
Built a 12-column slide grid with defined layout variants (cover, data, process, quote, next steps). Developed a brand-consistent color system, type scale, and modular components that any stakeholder can assemble without design support.
Deliverables
Full PowerPoint template system · Slide component library · Brand color & type guide · Scalable infographic and chart styles · Presenter notes framework
